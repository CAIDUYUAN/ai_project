방법 A · 구글 드라이브 자동 연동
매월 파일 올리면 자동으로 불러옴 · 초기 설정 1회 필요
▼
1
Apps Script 코드 등록
script.google.com 접속 → 새 프로젝트 생성 → 기본 코드 전체 삭제 후 아래 코드 붙여넣기 → 저장(Ctrl+S)
※ 코드는 이 대시보드 제작자에게 받으세요
2
Drive API v2 추가
왼쪽 메뉴 서비스(+) 클릭 → Google Drive API 선택 → 버전 v2 선택 → 추가
3
권한 승인 (최초 1회)
상단 함수 선택창에서 authorizeAll 선택 → ▶ 실행 → 팝업에서 Google 계정 로그인 및 권한 허용
4
웹앱으로 배포
우측 상단 배포 → 새 배포 → 유형: 웹앱 → 실행 계정: 나 · 액세스 권한: 모든 사용자 → 배포 → 생성된 웹앱 URL 복사
5
URL 입력
복사한 URL을 아래 Apps Script 배포 URL 칸에 붙여넣기
6
구글 드라이브 폴더 준비
드라이브에 전용 폴더 하나 생성 → 폴더 URL 아래 칸에 붙여넣기
※ 폴더는 본인 계정 소유이면 됩니다 (공유 불필요)
7
매출 파일 올리기 · 구글 시트로 변환
플랫폼에서 다운받은 .xlsx 파일을 폴더에 업로드 → 파일 우클릭 → 연결 앱 → Google 스프레드시트로 열기 → 파일 → Google 스프레드시트로 저장
⚡ 꿀팁: 여러 파일 한번에 변환하려면 Apps Script에서 convertAllExcelToSheets 함수 실행! (새 파일 추가할 때마다 1회 실행)
!
파일명 규칙 (변환 후 구글 시트 이름)
· 배민: 매출상세내역_YYYY년MM월...
· 쿠팡: coupang_eats_YYYY-MM
· 땡겨요: 땡겨요 YYYY년 MM월 매출 / 땡겨요 YYYY년 MM월 매입
· 요기요: 사업자번호_매출내역_YYYYMMDD_YYYYMMDD / 사업자번호_매입내역_YYYYMMDD_YYYYMMDD
파일명이 규칙과 다르면 플랫폼 자동 인식이 안 됩니다
✦ 선택사항 · 중복 파일 정리
+
중복 파일 자동 삭제 (파일명이 같은 파일 — 구글시트·xlsx 동일 취급)
Apps Script 편집기에서 함수 선택창 → removeDuplicates 선택 → ▶ 실행
⚡ 완전 삭제가 아닌 휴지통으로 이동 — 실수해도 복구 가능
구글 시트를 우선 남기고, 같은 형식이면 먼저 만들어진 파일 유지
실행 후 로그(Ctrl+Enter)에서 삭제된 파일 목록 확인 가능

방법 B · 엑셀 파일 직접 업로드
설정 없이 바로 사용 · 매번 파일 선택 필요
▼
1
배달의민족 파일 다운로드 (매출 + 매입)
배민 사장님사이트(self.baemin.com) → 정산 → 매출/매입 상세내역 → 월 선택 → 엑셀 다운로드
파일명 예시: 매출상세내역_2026년01월_... + 매입상세내역_2026년01월_...
2
쿠팡이츠 파일 다운로드
쿠팡이츠 사장님사이트 → 정산 → 매출 내역 → 월 선택 → 엑셀 다운로드
파일명 예시: coupang_eats_2026-03.xlsx
3
땡겨요 파일 다운로드 (매출 + 매입)
땡겨요 사장님사이트 → 부가세 신고자료 → 월 선택 → 매출/매입 각각 다운로드
파일명 예시: 땡겨요 2026년 03월 매출.xlsx + 땡겨요 2026년 03월 매입.xls
4
요기요 파일 다운로드 (매출 + 매입)
요기요 사장님사이트 → 부가세 신고자료 → 월 선택 → 매출/매입 각각 다운로드
파일명 예시: 227-31-01549_매출내역_20260101_20260131.xlsx + 227-31-01549_매입내역_20260101_20260131.xlsx
5
아래 박스에 파일 드래그 or 클릭
배민 / 쿠팡 / 땡겨요 / 요기요 각각의 박스에 업로드
여러 달 파일을 한번에 선택해도 됩니다
